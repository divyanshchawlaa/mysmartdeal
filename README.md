Full Stack E-Commerce Web Application built using the **MERN Stack**  
(MongoDB, Express, React, Node.js)

This project was developed as part of a Web Development module assignment.

---

## 🚀 Features

✅ Product listing  
✅ Product details page  
✅ Search products  
✅ Shopping cart  
✅ User login & register  
✅ Order history  
✅ PayPal sandbox payment  
✅ Admin product management  
✅ MongoDB database  
✅ Docker support  
✅ Responsive UI  
✅ Company info pages (About, Contact, FAQ, Privacy)

## ⚙️ Installation

Clone repository


git clone https://github.com/divyanshchawlaa/mysmartdeal

cd mysmartdeal


Install dependencies


npm install
cd frontend
npm install


Run backend


npm run server


Run frontend


npm start


Open browser


http://localhost:3000


---

## 💳 PayPal Sandbox

Payment is simulated using PayPal sandbox.


PAYPAL_CLIENT_ID=sb


No real payment is used.

---

## 🐳 Docker

Build container


docker build -t mysmartdeal .


Run container


docker run -p 5001:5001 mysmartdeal
